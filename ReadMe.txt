具体过程：
         魔方正对我们时，拍照面是橙色面，然后顺时针转，依次是蓝色，红色，绿色。拍到绿色面时，蓝色正对我们。然后向黄色方向转90度，拍取黄色面，然后同方向转，拍白色面。
